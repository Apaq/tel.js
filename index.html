Welcome to Tel.js

This is a simple demo of tel.js. The following input field is of type 'tel' and is therefore automatically formatted as a phone number. You can type in any valid phone number with countrycode(fx. +45 for Denmark) and it will automatically format it according to the country's formatting tradition. The phone number will also be validated so you can take advantage of the in Angulars forms.
